A modern, responsive personal portfolio designed to showcase my projects, skills, and professional journey. 

## 🚀 Technologies

* **Framework:** Angular (v17+) & TypeScript
* **Styling:** TailwindCSS (Utility-first, fully responsive)
* **Environment:** Docker (Isolated production builds)

## ⚙️ DevOps & CI/CD Pipeline

This project goes beyond frontend development by implementing a real-world **CI/CD pipeline**. 

* **Local Build:** Compilation is handled inside a **Docker** container to ensure environment consistency.
* **Automated Deployment:** Integrated with **GitHub Actions**. Every `git push` to the `main` branch triggers an automated workflow that extracts the production-ready build and deploys it seamlessly to GitHub Pages.
